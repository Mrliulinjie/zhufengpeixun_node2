# 安装包
- 安装jquery
```
    npm install jquery
```
- 指定版本安装
```
    npm install jquery@1.11.0
```
- 增加package.json文件里的开发依赖
```
    npm install jquery --save-dev
```
- 增加package.json文件里的依赖
```
    npm install jquery --save
```
- --save和--save-dev的区别
    - 开发的时候用到gulp，上线的时候用不到所以算是开发依赖 --save-dev
    - 就是我们的依赖--save
